# MorphPlayerShops
▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬

## Information:

A Minecraft plugin that allows players to set warp-able player shop locations and advertise them!
> https://www.spigotmc.org/resources/morphshops-1-13-x-1-16-x.59678/

## Build Information:

> I use maven as a dependency manager. You can simply run `mvn clean package` in the root of the project.
> The jar will be within the `Target` folder. If you run into any issues feel free to contact me.
